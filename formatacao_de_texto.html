Tag's

<P> </P>

A Tag P é a sigla para parágrafo, e serve para uso do texto puro.

H's

A tag H é a Tag para titulos, e quanto menor o número do H, maior o titulo. É recomendado que a Tag <H1> seja usada somente uma vez por documento.

<Br>

A tag BR é a tag sigla para Breakline.

<Sub> </Sub> e <Sup> <Sup>

A tag Sub é usada para colocar números ou letras abaixo da linha. ex: teste123

A tag Sup é usada para colocar números ou letras acima da linha. ex: teste123
